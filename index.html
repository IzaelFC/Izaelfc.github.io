Projetos

Este projeto é um site que ao ser acessado, busca na api da Unsplash imagens aleatórias e as organiza utilizando layout grid onde, ao clicar em uma foto sera aberto uma nova aba com a imagem original.

Famosa pokedex, onde se colaca o número do pokémon e será retornado as informaçoes do mesmo como o nome, habilidade, tipo(s) e ataques.

Uma página estática utilizando Tailwind na qual foi feita com base no desafio boraCodar #16 da Rockeseat.

Apenas uma página para compartilhar alguns poemas pessoais.
